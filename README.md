# Invisible Queue System

A Streamlit-powered queuing theory simulation showcasing an interactive, Tycoon-style airport check-in visualization.

This project visualizes abstract queue simulation mathematics using pure SVG, CSS keyframes, and real-time congestion algorithms—without any external JavaScript libraries.

## Features
- Scalable SVG UI mapped dynamically against mathematical backend arrays.
- Interactive queue length visualizations rendering entry/exit animations flawlessly on 1Hz data streams.
- Dynamic environmental congestion styling adapting visually against live Utilisation (ρ) metrics!
- Python backend handling core queue tracking seamlessly with pytest suite integrated.

## Stack
- Python
- Streamlit
- Custom HTML/SVG/CSS
